# ImageManipulatorAPI

This repository contains the ImageManipulator API that performs image operations (resize, rotate, compress, watermark, grayscale).

## Design

![Design](design.png)

## Quick Start

- Open the solution in Visual Studio: `ImageConverterBackend.sln`
- Run the `ImageConverterBackend` project.

## Notes

- The `design.png` file referenced above is located at the project root.
- For API details see the `Controllers` folder and the `Services` implementations.
